仕様と特徴

  1. 初心者向けのシンプルな UI で，日本語に完全対応しています．
  2. エディタ部分は Monaco Editor の v0.32.0 です．VS Code でも使用されている使いやすい環境です．シンタックスハイライト，対応する括弧の色付け，非 ASCII 文字のハイライトなどを設定しています．
  3. コード部分に使用しているフォントは UDEV Gothic です．可読性に優れた等幅フォントとなっています．
  4. Python の処理系は Pyodide の Version 0.21.0 です．Python のバージョンは 3.10.2 です．NumPy に標準で対応しているほか，micropip で各種モジュールを追加インストールすることも可能です．
  5. コードは 1 文字入力するたびにブラウザ内のローカルストレージに保存されます．ブラウザを誤って閉じてしまっても再び開けば途中から実装できます．
  6. 標準入力を受け取るための複雑なコードはワンクリックで挿入できます．ハイライトもされるため，誤って部分的に消してしまってもすぐに気づけます．
  7. 実行ボタン押下時に black によるオートフォーマットを実行します．初心者でも美しく見やすいコードを書けます．
  8. 実装中にもリアルタイムでエラーを表示します．実行時のエラーも検出します．
  9. 実行時の基本的なエラーメッセージは日本語で表示されます．さらに「なぜ間違っているか」だけでなく「どうすれば直せるか」まで提示する場合もあります．
  10. 無限ループの発生時も 10 秒経てば自動でタイムアウトします．
