PyTry の特徴

日本語エラー文

エラーメッセージが日本語で表示されます．一部のヒントにも対応しており，自分でバグを修正できます．

自動保存

ソースコードが自動でブラウザに保存されるので，いつでも実装を中断・再開できます．

オートフォーマット

ソースコードの見た目が自動で調整されるので，誰でも綺麗に実装できます．

コード挿入

標準入力を受け取るための複雑なコードは，ワンクリックで挿入できます．

詳細な仕様

  1. エディタは Monaco Editor の v0.32.0 です (VS Code と同じ環境です)．シンタックスハイライト，対応する括弧の色付け，非 ASCII 文字のハイライトなどを設定しています．
  2. コード部分に使用しているフォントは UDEV Gothic です．
  3. Python の処理系は Pyodide の Version 0.21.0 です．Python のバージョンは 3.10.2 です．NumPy に標準で対応しているほか，micropip で各種モジュールを追加インストールすることも可能です．
  4. ソースコードは 1 文字入力するたびにブラウザ内のローカルストレージに保存されます．
  5. 標準入力を受け取るための複雑なコードはワンクリックで挿入できます．削除防止のためハイライトされます．
  6. 実行ボタン押下時に black によるオートフォーマットを実行します．
  7. 実装中にもリアルタイムでエラーを表示します．実行時のエラーも検出します．
